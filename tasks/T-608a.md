### Contratos (T-608 §1, menos IntentNode)
```tsx
// --- apps/bancada/src/components/tabs/DadosTab.tsx ---
import { EntityId } from '@plataforma/protocol'; // T-102

export interface LinhagemInfo {
  entityId: EntityId;
  headId: EntityId;
  mutatesChainLength: number;
  forks: EntityId[];
  merges: EntityId[];
  frozen: boolean;
}

export interface DadosTabProps {
  linhagens: LinhagemInfo[];
  onForceFork: (entityId: EntityId) => void;
  onKillValidator: (entityId: EntityId) => void;
  onResolveFork: (forkEntityId: EntityId) => void;
}

export function DadosTab(props: DadosTabProps): JSX.Element;
```
- Navegador de linhagens: lista cada linhagem com cadeia MUTATES (profundidade), head atual, forks
  pendentes, indicação de congelamento.
- Botão "forçar fork" → `onForceFork` para a linhagem selecionada.
- Botão "derrubar validador" → `onKillValidator` (simula ausência de validador).
- Visualização de forks: expande para ver ramos e estado de resolução.
- **Extensibilidade:** deixe espaço estrutural (aba/section) para o inspetor de intents (T-608b)
  sem acoplar tipo — NÃO declare placeholder de `IntentNode`.
