## 1. Objetivo
Implementar aba "Dados" no app Bancada (T-008): navegador de linhagens (cadeia MUTATES, heads, forks pendentes, merges), inspetor de intents (pending/finalized, aprovações), botão "forçar fork" e "derrubar validador"
*(derivado de `docs/plano-de-implementacao.md` §9 T-608)*.

### Contratos
```tsx
// --- apps/bancada/src/tabs/DadosTab.tsx ---
import { EntityId } from '@plataforma/protocol'; // T-102
import { IntentNode } from '@plataforma/core'; // T-605

export interface LinhagemInfo {
  entityId: EntityId;
  headId: EntityId;
  mutatesChainLength: number;
  forks: EntityId[];
  merges: EntityId[];
  frozen: boolean;
}

export interface DadosTabProps {
  linhagens: LinhagemInfo[];
  intents: IntentNode[];
  onForceFork: (entityId: EntityId) => void;
  onKillValidator: (entityId: EntityId) => void;
  onResolveFork: (forkEntityId: EntityId) => void;
}

export function DadosTab(props: DadosTabProps): JSX.Element;
```
- Navegador de linhagens: lista cada linhagem com cadeia MUTATES (profundidade), head atual, forks pendentes, indicação de congelamento.
- Inspetor de intents: lista intents com estado (pending/finalized/rejected), número de aprovações, entidade alvo.
- Botão "forçar fork" → chama `onForceFork` para a linhagem selecionada.
- Botão "derrubar validador" → chama `onKillValidator` (simula ausência de validador).
- Visualização de forks: expande para ver ramos e estado de resolução.
