### Contratos exatos (contingentes — ver D1–D3)
```ts
// --- packages/core/src/sync/deterministicCommitter.ts ---
import type { EntityId } from '@plataforma/protocol'; // T-102 (derivado)
import type { Ed25519PrivateKey, Ed25519Signature } from '@plataforma/crypto'; // T-101a (derivado)
// ⚠️ import de CommitCycle (T-602) PENDE: pacote/local indefinido enquanto T-602 = pending_decision

export interface CommitCandidate {
  entityId: EntityId;            // document/linhagem
  changes: Uint8Array[];         // changes a commitar
  authorPubKey: Uint8Array;
  hlc: number;
}

export interface DeterministicTiebreak {
  /** Desempate determinístico: menor hash(entity_id ‖ HLC) vence. */
  winner: EntityId;
  reason: 'lower_hash' | 'higher_hlc' | 'single_author';
}

export interface CoSignRequest {
  commitId: string;
  requiredCoSigners: EntityId[]; // spec-driven
  signatures: Map<EntityId, Ed25519Signature>;
}

export interface CommitterResult {
  commitNodeId: EntityId;
  tiebreak: DeterministicTiebreak;
  coSignatures: CoSignRequest | null; // null se não exige co-assinatura
}

export class DeterministicCommitter {
  /** Ordena candidatos deterministicamente. */
  rankCandidates(candidates: CommitCandidate[]): CommitCandidate[];

  /** Executa commit com desempate determinístico. */
  commit(candidates: CommitCandidate[], options?: { requireCoSign?: boolean }): Promise<CommitterResult>;

  /** Solicita co-assinatura de peers via ephemeral channel. */
  requestCoSignature(request: CoSignRequest): Promise<void>;

  /** Verifica se co-assinatura está completa. */
  isCoSignComplete(request: CoSignRequest): boolean;
}
```
- `rankCandidates`: ordena por `hash(entity_id ‖ HLC)`; menor hash vence.
- Desempate determinístico: não depende de ordem de recebimento ou latência de rede.
- Co-assinatura: opcional, exigida pela SPEC para determinados tipos de entidade; via ephemeral (T-403).
- N peers commitando simultaneamente → não geram fork no caso comum (comutativo).
