## 1. Objetivo
Implementar `exportSnapshot` + `encodeSnapshot` no pacote `@plataforma/core`: extrai todos os nós e arestas do SQLite e serializa como binário MessagePack
*(extraído de T-308 §1)*.

### Contratos exatos
```ts
// --- packages/core/src/snapshot.ts ---
import { StoragePort } from '@plataforma/protocol'; // T-004

export interface Snapshot {
  version: number;
  createdAt: number;
  nodeCount: number;
  edgeCount: number;
  nodes: Uint8Array[];
  edges: Uint8Array[];
}

export function exportSnapshot(storage: StoragePort): Promise<Snapshot>;
export function encodeSnapshot(snapshot: Snapshot): Uint8Array;
```
