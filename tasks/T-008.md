## 1. Objetivo
Inicializar a "Bancada", que será a UI puramente de visualização para o sistema P2P. É um PWA em Vite + React que servirá como cockpit de diagnóstico. **Importante:** A Bancada (Fase B) não possui lógica de protocolo em si; ela atua como um "cliente burro" consumindo as rotas `/control` do Hub (`system-peer`) criadas na Fase A (T-009/T-010).
