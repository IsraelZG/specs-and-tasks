- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (Node puro)

## 1. Objetivo
Implementar Onda 3 e Garbage Collection v0 no pacote `@plataforma/core`: reidratação lazy de payloads podados via `REQUEST_NODES`/blobs sob demanda; GC com pools (LRU para grafo, rarest-first p/ blobs), pins (`ASSET:PIN`), pausa tier-aware por bateria, e protocolo de poda segura (jitter + RELEASE/ACK + health-check) — confirmação de N≥3 usa Archive Cargo (T-313) como mecanismo de custódia quando audiência < N
*(derivado de `docs/plano-de-implementacao.md` §11 T-806)*.

### Contratos exatos
```ts
// --- packages/core/src/sync/wave3.ts ---
import { EntityId, ULID } from '@plataforma/protocol'; // T-102
import { Fingerprint } from '@plataforma/protocol'; // T-301a
import { StoragePort } from '@plataforma/protocol'; // T-004

export enum GcPool {
  LRU_GRAPH = 'lru_graph',          // LRU para nós do grafo
  RAREST_FIRST_BLOBS = 'rarest_blobs', // rarest-first para blobs
}

export interface PinEntry {
  entityId: EntityId;
  pinnedBy: EntityId;
  pinnedAt: number;
  ttlMs?: number;                // TTL opcional para unpin automático
}

export interface PruneProtocol {
  jitterMs: number;              // jitter antes de podar (evita thundering herd)
  release: boolean;              // sinal RELEASE enviado
  ackRequired: number;           // número de ACKs esperados (default N≥3)
  healthCheck: boolean;          // verifica saúde dos custodiantes antes de podar
}

export interface Wave3Result {
  payloadsRehydrated: number;
  blobsRehydrated: number;
  nodesPruned: number;
  blobsPruned: number;
  archiveCargoUsed: boolean;    // se Archive Cargo foi usado (audiência < N)
}

export interface Wave3Gc {
  /** Onda 3: reidratação lazy de payloads podados. */
  executeWave3(session: RBSRSession): Promise<Wave3Result>;

  /** Reidrata payload sob demanda via REQUEST_NODES. */
  rehydratePayload(entityId: EntityId): Promise<Uint8Array>;

  /** Reidrata blob sob demanda. */
  rehydrateBlob(infoHash: string): Promise<Uint8Array[]>;

  /** GC: LRU do grafo. */
  pruneLruGraph(maxNodes: number): Promise<number>;

  /** GC: rarest-first para blobs. */
  pruneRarestFirstBlobs(minReplicas: number): Promise<number>;

  /** Pin: evita que entidade seja podada. */
  pin(entityId: EntityId): Promise<void>;

  /** Unpin: permite poda novamente. */
  unpin(entityId: EntityId): Promise<void>;

  /** Poda segura: jitter + RELEASE + ACK + health-check. */
  safePrune(entityIds: EntityId[], protocol: PruneProtocol): Promise<number>;

  /** Pausa tier-aware: suspende GC se bateria baixa. */
  pauseTierAware(batteryLevel: number): boolean;
}
```
- Onda 3: reidratação lazy de payloads podados — busca chunks/blobs sob demanda.
- GC LRU: remove nós menos recentemente usados do grafo local quando acima do limite.
- GC rarest-first: remove blobs que têm mais réplicas (preserva os raros).
- Pin: `ASSET:PIN` impede poda de entidades específicas.
- Poda segura: jitter aleatório + envia RELEASE + espera N ACKs + health-check dos custodiantes.
- Archive Cargo (T-313): usado como custódia quando audiência < N=3.
