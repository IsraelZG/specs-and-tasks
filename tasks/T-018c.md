## 1. Objetivo
Escrever testes unitários para **8 componentes de overlay/disclosure** do `@plataforma/design-system`, cobrindo os critérios do Tier B (smoke fechado, abrir com role, fechar via interação, ref).

> **PRÉ-REQUISITO:** ambiente de teste do design-system deve existir (ver T-018 Seção 6). O "Modal.test.tsx" da spec original foi refutado no disco.

### Componentes cobertos por esta task
1. `AlertDialog`
2. `Command`
3. `DropdownMenu`
4. `HoverCard`
5. `Menubar`
6. `Popover`
7. `Sheet`
8. `Tooltip`

### Critérios de aceite (Tier B — overlay/disclosure)
1. **Smoke fechado:** `queryByRole` retorna `null` quando fechado.
2. **Abrir overlay:** via prop controlada `open`/`isOpen` ou interação, conforme a API. Conteúdo renderiza com o role esperado (`dialog`, `menu`, `tooltip`, etc.) e atributo `aria-*` correspondente.
3. **Fechar:** via interação documentada (clique fora, Escape, clique no trigger) dispara callback de fechamento.
4. **Ref forwarding:** se `React.forwardRef`, ref chega ao nó DOM esperado.
