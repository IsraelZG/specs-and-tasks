## 1. Objetivo
Implementar aba "Identidade" no app Bancada (T-008): criar/importar mnemônico BIP39, exibir DevicePeerId/PersonaPeerId, travar/destravar com senha, reset de identidade
*(derivado de `docs/plano-de-implementacao.md` M1 — T-111)*.

### Contratos exatos
```tsx
// --- apps/bancada/src/tabs/IdentidadeTab.tsx ---
export function IdentidadeTab(): JSX.Element;
```
Funcionalidades:
- Botão "Gerar Identidade" → chama `generateMnemonic()` (T-104a), exibe palavras.
- Campo "Importar Mnemônico" → `validateMnemonic()`, armazena.
- Exibição: DevicePeerId, PersonaPeerId (truncados).
- Botão "Travar" / "Destravar" → solicita senha via prompt.
- Botão "Reset" → confirmação + `resetPeer()` (T-017).
