## 1. Objetivo
Implementar detecção estrutural de fork + merge determinístico no pacote `@plataforma/core`: identifica quando o DAG bifurca (2 arestas MUTATES concorrentes na mesma linhagem) e aplica resolução por HLC máximo + tie-break por `author_pubkey`
*(derivado de `docs/conceitos/hlc.md` — "detecção de fork é estrutural"; T-108 — linhagem Layer 2; T-302 — RBSR exchange traz nós divergentes)*.

### Contratos
```ts
// --- packages/core/src/dag/merge.ts ---
import { SignedNode } from '../signature.js'; // T-107
import { StoragePort } from '@plataforma/protocol'; // T-004

/** Detecta fork: duas ou mais arestas MUTATES ativas com mesmo sourceId. */
export function detectForks(storage: StoragePort, sourceId: string): Promise<SignedNode[]>;

/** Resolve fork deterministicamente: nó com maior HLC vence; empate → maior pubkey lexicográfica. */
export function resolveFork(forks: SignedNode[]): SignedNode;

/** Cria nó de merge apontando para as duas pontas do fork. */
export function createMergeNode(
  forkA: SignedNode,
  forkB: SignedNode,
  authorKey: Ed25519PrivateKey,
): Promise<SignedNode>;

/** Aplica merge: insere nó-merge no grafo via insertNode de T-108. */
export function applyMerge(storage: StoragePort, forkA: SignedNode, forkB: SignedNode): Promise<void>;
```
- Fork detection: `SELECT * FROM edges WHERE edge_type='MUTATES' AND source_id=?` com ≥2 resultados.
- Resolução: HLC máximo + `author_pubkey` como tie-break (determinístico, idêntico em todos os peers).
- Merge: nó apontando `MUTATES` para ambas as pontas, com `entity_id` herdado.

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro). SQLite :memory: + SimNetwork para simular partição.

Casos de teste:
1. 2 nós concorrentes na mesma linhagem → `detectForks` retorna ambos.
2. `resolveFork`: nó com HLC maior vence.
3. `resolveFork`: HLC igual → `author_pubkey` maior vence.
4. `createMergeNode` gera nó com 2 arestas MUTATES.
5. SimNetwork: particiona → cada peer edita → heal → merge converge para mesmo estado.
