## 1. Objetivo
Implementar Archive Cargo no pacote `@plataforma/core`: empacotamento de lote por época (zstd + AES-256-GCM), `archive_id = blake2s256(ciphertext)`, manifesto `CONTENT` sob `SPECIFICATION:ARCHIVE_MANIFEST`, arestas `ARCHIVES` sob sync UCAN-scoped, entrega via canal `EPHEMERAL`, armazenamento em `device_state.db` (tabela `blind_archives`), atribuição por consistent hashing sobre `blind_scope_id`, GC por `expires_at`
*(derivado de `docs/plano-de-implementacao.md` §6 T-313 — "RFC-005 §A.11")*.

### Contratos exatos
```ts
// --- packages/core/src/archive/archiveCargo.ts ---
import { EntityId, ULID } from '@plataforma/protocol'; // T-102
import { StoragePort } from '@plataforma/protocol'; // T-004
import { Fingerprint } from '@plataforma/protocol'; // T-301a

export interface ArchiveCargoPayload {
  blindScopeId: Uint8Array;       // scope cego (SHA-256 do entity_id real)
  epochIndex: number;
  hlcRange: { start: number; end: number };
  size: number;                   // bytes do ciphertext
  expiresAt: number;              // timestamp Unix ms
}

export interface ArchiveManifest {
  archiveId: Fingerprint;         // blake2s256(ciphertext)
  blindScopeId: Uint8Array;
  epochIndex: number;
  hlcRange: { start: number; end: number };
  size: number;
  expiresAt: number;
}

export interface ArchiveCargo {
  ciphertext: Uint8Array;         // zstd + AES-256-GCM com chave de época do escopo
  manifest: ArchiveManifest;
}

export interface ArchiveConfig {
  epochKey: Uint8Array;           // chave AES-256 da época do escopo
  compressionLevel?: number;      // zstd (default 3)
}

/** Empacota lote de nós + arestas em Archive Cargo. */
export function packArchive(
  nodes: Uint8Array[],            // nós serializados
  edges: Uint8Array[],            // arestas serializadas
  epochIndex: number,
  entityId: EntityId,
  config: ArchiveConfig,
): Promise<ArchiveCargo>;

/** Desempacota e decifra Archive Cargo. */
export function unpackArchive(cargo: ArchiveCargo, config: ArchiveConfig): Promise<{ nodes: Uint8Array[]; edges: Uint8Array[] }>;

/** Atribui custodiante por consistent hashing sobre blind_scope_id. */
export function assignCustodian(blindScopeId: Uint8Array, custodians: EntityId[]): EntityId;

/** Verifica integridade self-certifying. */
export function verifyArchiveIntegrity(cargo: ArchiveCargo): boolean;

/** GC: remove archives expirados. */
export function garbageCollectArchives(storage: StoragePort): Promise<number>;
```
- `packArchive`: concatena nós+arestas → zstd → AES-256-GCM (key=epochKey, nonce aleatório) → `archiveId = blake2s256(ciphertext)`.
- `unpackArchive`: decifra → descomprime → retorna arrays originais.
- `verifyArchiveIntegrity`: confere `blake2s256(ciphertext) === archiveId` + tag GCM válida.
- Custodiante: `consistent hashing` sobre `blind_scope_id` para distribuir archives entre peers.
- GC: remove da tabela `blind_archives` onde `expires_at < now()`.
- Vetor adversarial 6 estendido: peer sem UCAN não enxerga arestas `ARCHIVES`.
