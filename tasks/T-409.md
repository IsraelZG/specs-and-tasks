## 1. Objetivo
Implementar aba "Topologia" no app Bancada (T-008): grafo visual de conexões (direta/relay/ws), estado de promoção por par, botão "forçar relay"/"tentar promoção", indicador de quem é âncora
*(derivado de `docs/plano-de-implementacao.md` §7 T-409)*.

### Contratos
```tsx
// --- apps/bancada/src/tabs/TopologiaTab.tsx 
---
import { SwarmRegistry, PeerRecord } from '@plataforma/core'; // T-205
import { ConnectionPromotionEngine } from '@plataforma/transport'; // T-404

export interface TopologiaTabProps {
  swarm: SwarmRegistry;
  promotionEngine: ConnectionPromotionEngine;
}

export function TopologiaTab({ swarm, promotionEngine }: TopologiaTabProps): JSX.Element;
```
- Renderiza grafo visual de conexões: nós = peers, arestas = conexões.
- Cada aresta colorida por tipo: verde = direta (WebRTC), azul = relay, cinza = WS.
- Indicador de âncora (peer do sistema) com ícone distinto.
- Estado de promoção por par: "direta", "promovendo...", "relay (NAT simétrico)".
- Botão "forçar relay" → migra conexão para relay.
- Botão "tentar promoção" → dispara hole punching.
- Atualização em tempo real via polling (5s) ou eventos do SwarmRegistry.
