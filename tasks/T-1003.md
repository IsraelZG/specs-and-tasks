# T-1003 · Roteamento Inteligente com SmolLM2 (llmlingua-js)

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v22+
- **Package Manager:** `pnpm`
- **Test Runner:** `vitest`

## 1. Objetivo
Integrar o pacote local `llmlingua-js` (SmolLM2 rodando via ONNX/WASM) no backend. O objetivo inicial é expor um serviço que lê uma "intenção" do usuário e decide, localmente e rápido, quais tópicos ou arquivos devem ser filtrados pelo TurboVec.

## 5. Instruções de Execução (Step-by-Step)
> **⚠️ REGRAS DO QUE NÃO FAZER:**
> - **NÃO** faça chamadas a APIs da OpenAI/Anthropic aqui. Toda inferência deve rodar no pacote local baseado em Transformers.js.

1. Adicione a dependência da sua versão local do `llmlingua-js`.
2. Instancie o modelo ONNX no startup do servidor backend.
3. Crie uma função `routeIntent(prompt)` que injeta o prompt do usuário no LLM local pedindo que ele retorne keywords ou tags de contexto relevantes.
