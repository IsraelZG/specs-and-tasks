# T-801 · Storage Engine de BLOBs (Chunking)

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm` (NÃO USE npm ou yarn)
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (pacotes core/protocol) e `playwright` (E2E/Frontend)

## 1. Objetivo
Gravar arquivos (fotos/vídeos) fragmentados no OPFS/IndexedDB, hasheados em uma Merkle Tree para sync descentralizado.
