# T-001 · Bootstrap do monorepo (Exemplo Gold Standard)

> **ATENÇÃO — o monorepo JÁ EXISTE.** Este repo já é um monorepo pnpm/turbo (a ferramenta `nexus`,
> que está **CONGELADA** — não a toque, ver nota no `CLAUDE.md`). Esta task **ADICIONA** os pacotes
> `@plataforma/*` + `tsconfig.base.json`; **NÃO recria** `package.json`/`pnpm-workspace.yaml`/

- **Runtime:** Node.js v20+

- **Monorepo:** Turborepo (já configurado). O build/lint da raiz já exclui o nexus congelado; rode o gate escopado em \`@plataforma/*\`.
- **Test Runner:** \`vitest\` (pacotes core/protocol) e \`playwright\` (E2E/Frontend)
- **Capacidade-alvo:** haiku
