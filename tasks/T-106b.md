- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo

## 1. Objetivo
Implementar `migrateSchema(storage)` no pacote `@plataforma/core`: aplica migrations pendentes via `StoragePort.migrate()`, idempotente, com tracking de versão
*(extraído de T-106 §1)*.

### Contratos exatos
```ts
// --- packages/core/src/schema.ts (extensão) ---

import { MIGRATIONS } from './schema.js'; // T-106a

/** Aplica migrations pendentes. Idempotente — não re-aplica versões já instaladas. */
export async function migrateSchema(storage: StoragePort): Promise<void>;
```
