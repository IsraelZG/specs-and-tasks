# T-704 · Bancada: simulador de segundo device

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (React Testing Library) / `playwright` (E2E)
- **Capacidade-alvo:** haiku

## 1. Objetivo
Implementar simulador de segundo device no app Bancada (T-008): botão "abrir como novo device da mesma identidade" (novo context com a mesma seed) + visualização do estado do private swarm
*(derivado de `docs/plano-de-implementacao.md` §10 T-704)*.

### Contratos
```tsx
// --- apps/bancada/src/tabs/DeviceSimTab.tsx ---
import { PrivateSwarm, PrivateSwarmChannel } from '@plataforma/transport'; // T-702

export interface DeviceSimTabProps {
  privateSwarm: PrivateSwarm;
  currentDeviceId: string;
  onOpenSecondDevice: () => void;
}

export function DeviceSimTab({ privateSwarm, currentDeviceId, onOpenSecondDevice }: DeviceSimTabProps): JSX.Element;
```
- Botão "Abrir como novo device": cria novo browser context com a mesma seed BIP39 → simula segundo device.
- Visualização do private swarm: lista devices conectados com last sync time.
- Estado de sync: mostra prefetch cache, rascunhos pendentes, preferências sincronizadas.
- Indicador de convergência entre devices.
- Botão "Forçar sync" para disparar reconciliação manual.
