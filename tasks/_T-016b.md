# T-016b · Tema Multi-Nível: themeEngine (compileScopedOverrides)

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo
- **Test Runner:** `vitest`
- **Capacidade-alvo:** sonnet

## 1. Objetivo
Estender `themeEngine.ts` no `packages/design-system` com `compileScopedOverrides` e `overridesToStyleObject`: gera CSS escopado (blocos `<style>`) e objetos de estilo inline a partir de dicionários de override, usando os tokens com `outputReferences: true` da T-016a
*(extraído de T-016 §1; `docs/rfcs/design-system-proposal.md` D1, D8)*.

### Contratos exatos
```ts
// --- packages/design-system/src/lib/themeEngine.ts (extensão) ---

export interface ThemeOverride {
  scope?: string;        // 'module' | 'page' | 'component'
  scopeId?: string;      // id do escopo
  tokens: Record<string, string>; // 'theme.intent.primary.fill': '#ff0000'
}

/** Gera bloco <style> com custom properties escopadas via [data-ds-<scope>="<id>"]. */
export function compileScopedOverrides(overrides: ThemeOverride[]): string;

/** Gera objeto de estilo inline { '--ds-theme-intent-primary-fill': '#ff0000' }. */
export function overridesToStyleObject(overrides: ThemeOverride[]): Record<string, string>;
```
Exemplo de output CSS:
```css
[data-ds-module="chat"] { --ds-theme-intent-primary-fill: #ff0000; }
[data-ds-page="settings"] { --ds-theme-intent-primary-fill: #00ff00; }
```
