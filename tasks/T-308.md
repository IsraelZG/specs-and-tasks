## 1. Objetivo
Implementar export/import de snapshot de bootstrap no pacote `@plataforma/core`: compacta nós + arestas do SQLite como binário MessagePack para acelerar onboarding de novos peers
*(derivado de `docs/plano-de-implementacao.md` §3 T-308; schema `nodes`/`edges` de T-106; wire format de T-201)*.

### Contratos exatos
```ts
// --- packages/core/src/snapshot.ts ---
import { StoragePort } from '@plataforma/protocol'; // T-004

/** Snapshot compactado: todos os nós e arestas serializados. */
export interface Snapshot {
  version: number;         // 1
  createdAt: number;       // timestamp ms (wall clock, informativo)
  nodeCount: number;
  edgeCount: number;
  nodes: Uint8Array[];     // cada nó = canonicalize + signature (binário)
  edges: Uint8Array[];     // cada aresta = canonicalize + signature (binário)
}

/** Exporta subgrafo completo do SQLite como snapshot binário. */
export function exportSnapshot(storage: StoragePort): Promise<Uint8Array>;

/** Importa snapshot no SQLite. Idempotente: ignora duplicatas por id. */
export function importSnapshot(storage: StoragePort, data: Uint8Array): Promise<{ importedNodes: number; importedEdges: number }>;

/** Serializa snapshot para MessagePack. */
export function encodeSnapshot(snapshot: Snapshot): Uint8Array;

/** Desserializa snapshot de MessagePack. */
export function decodeSnapshot(data: Uint8Array): Snapshot;
```
- `exportSnapshot`: `SELECT * FROM nodes` + `SELECT * FROM edges` empacotados como `Snapshot`.
- `importSnapshot`: `INSERT OR IGNORE` para cada nó/aresta (idempotente).
- Serialização: MessagePack (msgpackr) dos arrays de bytes.

Casos de teste:
1. `exportSnapshot` sobre DB vazio → snapshot com 0 nodes, 0 edges.
2. Insere 3 nós + 2 arestas → `exportSnapshot` retorna buffer não vazio.
3. `decodeSnapshot(encodeSnapshot(s))` → roundtrip estrutural.
4. `importSnapshot` em DB limpo → dados recuperados via `SELECT`.
5. `importSnapshot` 2× (idempotência) → sem duplicatas, contagem igual.
6. Snapshot com 1000 nós → `importSnapshot` < 500ms.
