### Contratos exatos (assinaturas TS fixadas)

```ts
// --- packages/core/src/signature.ts ---
import { Ed25519PublicKey, Ed25519PrivateKey, Ed25519Signature } from '@plataforma/crypto'; // T-101
import { ULID } from './ulid.js'; // T-102
import { HLCTimestamp } from './hlc.js'; // T-103

/** Nó do grafo sem assinatura (pré-Layer 1). */
export interface UnsignedNode {
  id: ULID;
  nodeType: string;           // 'PROFILE' | 'ASSET' | 'SPECIFICATION' | 'CONTENT'
  payload: Uint8Array;
  hlc: HLCTimestamp;
  publicKey: Ed25519PublicKey;
}

/** Nó do grafo com assinatura Layer 1. */
export interface SignedNode extends UnsignedNode {
  signature: Ed25519Signature;
}

/** Aresta do grafo sem assinatura. */
export interface UnsignedEdge {
  id: ULID;
  edgeType: string;           // 'MUTATES' | 'DELEGATED_TO' | ...
  sourceId: ULID;
  targetId: ULID;
  payload: Uint8Array | null;
  hlc: HLCTimestamp;
  publicKey: Ed25519PublicKey;
}

/** Aresta do grafo com assinatura Layer 1. */
export interface SignedEdge extends UnsignedEdge {
  signature: Ed25519Signature;
}

/** Serialização canônica para assinatura: concatena campos em ordem léxica de nome.
 *  Nó: hlc(8B BE) | id(26B string utf-8) | nodeType_len(u8) | nodeType | payload_len(u32BE) | payload | publicKey(32B)
 *  Aresta: edgeType_len(u8) | edgeType | hlc(8B BE) | id(26B string utf-8) | payload_len(u32BE) | payload | publicKey(32B) | sourceId(26B string utf-8) | targetId(26B string utf-8)
 *  Campos são concatenados como bytes, NÃO como MessagePack/JSON. */
export function canonicalizeNode(node: UnsignedNode): Uint8Array;

export function canonicalizeEdge(edge: UnsignedEdge): Uint8Array;

/** Assina nó com chave privada. */
export function signNode(node: UnsignedNode, privateKey: Ed25519PrivateKey): Promise<SignedNode>;

/** Assina aresta. */
export function signEdge(edge: UnsignedEdge, privateKey: Ed25519PrivateKey): Promise<SignedEdge>;

/** Verifica assinatura de nó. */
export function verifyNode(node: SignedNode): Promise<boolean>;

/** Verifica assinatura de aresta. */
export function verifyEdge(edge: SignedEdge): Promise<boolean>;
```

### Regras de canonicidade (derivado de visao-arquitetural.md §5)
- Ordem dos campos no buffer: **alfabética por nome** (edgeType, hlc, id, nodeType, payload, publicKey, ...).
- `payload` pode ser `null` (arestas sem payload) → `payload_len = 0`.
- A assinatura cobre **apenas** o `canonicalize`, não o framing de rede (T-201 §3 item 4).

Casos de teste (numerados):
1. `signNode` + `verifyNode` roundtrip → true.
2. `verifyNode` com assinatura de chave pública errada → false.
3. `verifyNode` com payload adulterado (1 byte trocado) → false.
4. `verifyNode` com hlc adulterado → false.
5. `canonicalizeNode`: dois nós com campos iguais mas ordem de inserção diferente no objeto produzem o mesmo buffer (ordem léxica de campo).
6. `signEdge` + `verifyEdge` roundtrip com payload → true.
7. `signEdge` + `verifyEdge` roundtrip com payload=null → true.
8. `verifyEdge` com sourceId trocado → false.
9. Dois nós com mesmo conteúdo mas ids diferentes → assinaturas diferentes.
10. `canonicalize` é determinístico: chamar 100× com mesma entrada produz mesmo Uint8Array.
