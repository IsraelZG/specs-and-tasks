## 1. Objetivo
Implementar geração e parse de link/QR com multiaddr no fragment no pacote `@plataforma/protocol`: primeiro contato via relay stateless; peer novo entra em rede P2P-pura por link, sem cloud
*(derivado de `docs/plano-de-implementacao.md` §7 T-407)*.

### Contratos exatos
```ts
// --- packages/protocol/src/outOfBand.ts ---
import { ParsedMultiaddr } from './peerId.js';
import { Ed25519PublicKey } from '@plataforma/crypto';

export interface OutOfBandLink {
  multiaddr: ParsedMultiaddr;          // multiaddr do peer remoto (parsed)
  ephemeralPubKey: Ed25519PublicKey; // chave pública efêmera p/ primeiro contato
  nonce: Uint8Array;             // 16 bytes — nonce do convite
  relayHint?: ParsedMultiaddr;   // multiaddr de relay stateless (opcional)
}

/** Gera link out-of-band a partir de multiaddr + chave efêmera. */
export function generateOobLink(
  multiaddr: ParsedMultiaddr,
  ephemeralPubKey: Ed25519PublicKey,
  relayHint?: ParsedMultiaddr,
): string;

/** Parse de link out-of-band → estrutura. */
export function parseOobLink(link: string): OutOfBandLink;

/** Gera representação QR-ready (compacta, uppercase base64url-safe). */
export function toQrPayload(link: OutOfBandLink): string;

/** Parse de QR payload → estrutura. */
export function fromQrPayload(qr: string): OutOfBandLink;
```
- `generateOobLink`: formato `plataforma://connect#multiaddr=<encoded>&ephemeral=<b64>&nonce=<b64>&relay=<encoded>`.
- `parseOobLink`: parse do formato URL com fragment.
- `toQrPayload` / `fromQrPayload`: versão compacta para QR code (limite ~2KB).
- Primeiro contato: peer remoto usa relay stateless para estabelecer conexão inicial.
