## 1. Objetivo
Implementar `MerkleTree` no pacote `@plataforma/media`: constrói árvore a partir dos hashes dos chunks, gera provas de inclusão e verifica provas
*(extraído de T-801 §1)*.

### Contratos exatos
```ts
// --- packages/media/src/blob/MerkleTree.ts ---
import { Chunk } from './Chunker.js'; // T-801a

export interface MerkleNode {
  hash: Uint8Array;
  left?: MerkleNode;
  right?: MerkleNode;
}

export interface MerkleProofStep {
  position: 'left' | 'right';
  hash: Uint8Array;
}

export interface MerkleProof {
  rootHash: Uint8Array;
  chunkIndex: number;
  proof: MerkleProofStep[];
}

/** Constrói Merkle Tree bottom-up. */
export function buildMerkleTree(chunks: Chunk[]): MerkleNode;

/** Gera prova para um chunk. */
export function generateProof(root: MerkleNode, chunks: Chunk[], index: number): MerkleProof;

/** Verifica prova contra rootHash. */
export function verifyProof(proof: MerkleProof, chunkHash: Uint8Array): boolean;
```

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro).

Casos de teste:
1. `buildMerkleTree` com 4 chunks → árvore com 3 níveis, root hash calculado.
2. `buildMerkleTree` com 1 chunk → root = hash do chunk.
3. `generateProof` + `verifyProof` roundtrip para chunk[0].
4. `verifyProof` com chunk adulterado → false.
5. `verifyProof` com rootHash errado → false.
6. 256 chunks → árvore balanceada, prova de profundidade O(log n).
