# T-1011 · Ferramentas MCP + endpoints REST de Tasks

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Test Runner:** `vitest`

## 1. Objetivo
Expor o `task.service` (T-1010) por dois canais com paridade: **MCP** (para os
agentes que consomem o Nexus) e **REST** (para a UI e como fallback). Ambos os
canais devem chamar exclusivamente o `task.service` — sem duplicar lógica de escrita.
