## 1. Objetivo
Implementar `DevicePeerId` e `PersonaPeerId` no pacote `@plataforma/protocol`, ambos derivados de chave pública Ed25519 via blake2s256
*(extraído de T-105 §1; derivado de `docs/conceitos/peer-id.md`)*.

### Contratos exatos
```ts
// --- packages/protocol/src/peerId.ts ---
import { Ed25519PublicKey } from '@plataforma/crypto'; // T-101a

export type DevicePeerId = Uint8Array;  // 32 bytes = blake2s256(DEVICE_PUB_KEY)
export type PersonaPeerId = Uint8Array; // 32 bytes = blake2s256(PERSONA_PUB_KEY)
export type PeerId = DevicePeerId | PersonaPeerId;

export function deriveDevicePeerId(devicePubKey: Ed25519PublicKey): DevicePeerId;
export function derivePersonaPeerId(personaPubKey: Ed25519PublicKey): PersonaPeerId;

/** Codifica PeerId (32 bytes) como string base58btc. */
export function peerIdToString(peerId: PeerId): string;

/** Decodifica string base58btc → PeerId. */
export function stringToPeerId(str: string): PeerId;
```

5. `stringToPeerId` com string inválida → erro.
