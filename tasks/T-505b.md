## 1. Objetivo
Implementar métodos de consulta no `EpochRotation`: `getCurrentEpoch`, `getEpoch(index)`, `listEpochs`
*(extraído de T-505 §1)*.

### Contratos exatos
```ts
// --- packages/core/src/vault/EpochRotation.ts (extensão) ---

export class EpochRotation {
  // ... rotate/revoke de T-505a

  getCurrentEpoch(): EpochConfig | null;
  getEpoch(index: number): EpochConfig | null;
  listEpochs(): EpochConfig[];
}
```
- `getCurrentEpoch`: retorna época com `currentIndex`, verificando TTL via `KeyVault.get`.
- `listEpochs`: todas as épocas ativas no vault.
