## 1. Objetivo
Implementar `noiseHandshakeInitiator` no pacote `@plataforma/transport`: inicia handshake Noise_XX (3 round-trips) como iniciador, usando `NetworkAdapterPort` para troca de mensagens. Troca DevicePeerId + identity_epoch_index + nonce assinado. Deriva chaves de sessão AES ao final
*(extraído de T-202 §1; `docs/conceitos/noise-xx.md` — "3 round-trips, autenticação mútua")*.

### Contratos exatos
```ts
// --- packages/transport/src/noise.ts ---
import { NetworkAdapterPort, PeerId } from '@plataforma/protocol'; // T-004
import { Ed25519PrivateKey, Ed25519PublicKey } from './ed25519.js'; // T-101a
import { DevicePeerId } from '@plataforma/protocol'; // T-105a

export enum NoiseState { HANDSHAKE = 'handshake', CONNECTED = 'connected', FAILED = 'failed' }

export interface NoiseResult {
  remoteDevicePeerId: DevicePeerId;
  remoteIdentityEpochIndex: number;
  encryptKey: Uint8Array;  // AES-256 chave de sessão (saída)
  decryptKey: Uint8Array;  // AES-256 chave de sessão (entrada)
}

/** Inicia handshake Noise_XX como iniciador. */
export function noiseHandshakeInitiator(
  deviceKeyPair: { publicKey: Ed25519PublicKey; privateKey: Ed25519PrivateKey },
  identityEpochIndex: number,
  transport: NetworkAdapterPort,
  remotePeerId: PeerId,
): Promise<NoiseResult>;
```
Fluxo (3 RT):
1. Initiator → ephemeral key
2. Responder → ephemeral + static key (com DevicePeerId + identity_epoch_index)
3. Initiator → static key (com DevicePeerId + identity_epoch_index + nonce assinado)
