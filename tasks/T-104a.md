# T-104a · BIP39: Mnemônico + Validação

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (Node puro)
- **Capacidade-alvo:** haiku

## 1. Objetivo
Implementar geração de mnemônico BIP39 (12/24 palavras), validação de checksum e conversão mnemonic↔entropy no pacote `@plataforma/crypto`
*(extraído de T-104 §1)*.

### Contratos exatos
```ts
// --- packages/crypto/src/bip39.ts ---
export type Entropy = Uint8Array; // 16B (12 palavras) ou 32B (24 palavras)

/** Gera mnemônico de 12 palavras (128 bits de entropia via WebCrypto). */
export function generateMnemonic(): string;

/** Gera mnemônico de 24 palavras (256 bits). */
export function generateMnemonic24(): string;

/** Valida checksum BIP39 (últimos bits da entropia conferem com SHA-256). */
export function validateMnemonic(mnemonic: string): boolean;

/** Converte mnemônico → entropy (16 ou 32 bytes). */
export function mnemonicToEntropy(mnemonic: string): Entropy;

/** Converte entropy → mnemônico (com checksum). */
export function entropyToMnemonic(entropy: Entropy): string;
```

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro).

Casos de teste:
1. `generateMnemonic()` → 12 palavras; `validateMnemonic` → true.
2. `generateMnemonic24()` → 24 palavras válidas.
3. `validateMnemonic` com palavra trocada → false.
4. `validateMnemonic` com checksum errado → false.
5. `entropyToMnemonic` + `mnemonicToEntropy` roundtrip (12 e 24 palavras).
6. BIP39 wordlist contém exatamente 2048 palavras.
