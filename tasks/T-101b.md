# T-101b · Wrapper AES-256-GCM + SHA-256

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (Node puro)
- **Capacidade-alvo:** haiku

## 1. Objetivo
Implementar wrappers isomórficos para AES-256-GCM (encrypt/decrypt) e SHA-256 no pacote `@plataforma/crypto`. AES usa WebCrypto (`crypto.subtle`); SHA-256 usa `@noble/hashes/sha256`
*(extraído de T-101 §1)*.

### Contratos exatos
```ts
// --- packages/crypto/src/aes.ts ---
export type AesKey = Uint8Array;   // 32 bytes
export type AesNonce = Uint8Array; // 12 bytes

/** Cifra com AES-256-GCM. Gera nonce internamente (12 bytes). */
export function aesGcmEncrypt(key: AesKey, plaintext: Uint8Array): Promise<{ nonce: AesNonce; ciphertext: Uint8Array; authTag: Uint8Array }>;

/** Decifra AES-256-GCM. Lança se authTag não confere. */
export function aesGcmDecrypt(key: AesKey, nonce: AesNonce, ciphertext: Uint8Array, authTag: Uint8Array): Promise<Uint8Array>;
```

```ts
// --- packages/crypto/src/sha256.ts ---
export type Sha256Digest = Uint8Array; // 32 bytes

export function sha256(data: Uint8Array): Promise<Sha256Digest>;
```

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro).
- [x] **Ambiente:** `pnpm --filter @plataforma/crypto test`.

Casos de teste:
1. `aesGcmEncrypt` + `aesGcmDecrypt` roundtrip (vazio, 1B, 1KB, 1MB).
2. `aesGcmDecrypt` com authTag corrompido → lança erro.
3. `aesGcmDecrypt` com ciphertext corrompido → lança erro.
4. `sha256` sobre `Uint8Array` vazio → RFC 6234 (e3b0c44298fc1c14...).
5. `sha256` determinístico: mesma entrada → mesmo digest.
6. `sha256` com 1MB → < 50ms.

## 5. Instruções de Execução
> **⚠️ REGRAS DO QUE NÃO FAZER:**
> - **NÃO** use `crypto.createCipheriv` do Node puro — AES deve usar WebCrypto (`crypto.subtle`) para isomorfismo.
> - **NÃO** implemente SHA-256 do zero — use `@noble/hashes/sha256`.

1. **[TDD]** Crie testes com casos 1–6.

3. Implemente `sha256` via `@noble/hashes/sha256`.
4. Re-exporte. Rode build + test.
