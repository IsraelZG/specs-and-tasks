## 0. Ambiente de Execução Obrigatório

## 1. Objetivo
Implementar gênese gerida no peer do sistema: inicialização com token de fundação via config, fundação de rede corporativa/pública e URL de onboarding
*(derivado de `docs/plano-de-implementacao.md` §5 T-209)*.

### Contratos exatos
```ts
// --- apps/system-peer/src/genesis.ts ---
export interface GenesisConfig {
  founderToken: string;          // bootstrap token do admin
  networkName: string;           // nome da rede
  multiaddrHost: string;         // host público p/ onboarding (ex: "host.com")
  multiaddrPort: number;         // porta WS
}

export interface GenesisResult {
  success: boolean;
  networkId: string;
  onboardingUrl: string;         // "https://host/sync#multiaddr=..."
  profileEntityId: string;       // PROFILE do admin
  birthEntityId: string;         // SPECIFICATION:NETWORK_BIRTH
}

export interface GenesisService {
  /** Funda a rede corporativa/pública. Idempotente: se já fundada, retorna dados existentes. */
  found(config: GenesisConfig): Promise<GenesisResult>;

  /** Verifica se a rede já foi fundada. */
  isFounded(): Promise<boolean>;

  /** Gera URL de onboarding para convidados. */
  generateOnboardingUrl(config: GenesisConfig): string;
}
```
- Token de fundação validado antes da gênese: `blake2s256(config.founderToken) === founderHash`.
- Criação: `PROFILE` admin + `SPECIFICATION` do workspace + `SPECIFICATION:NETWORK_BIRTH` imutável + flag `PROVISIONAL_SYSTEM_PEER`.
- URL de onboarding: `https://<host>/sync#multiaddr=<encoded>` (fragment, não query).
- Transição a peer normal na chegada do segundo peer (FPP T-208).

Casos de teste:
1. `found` com token válido → rede fundada, `onboardingUrl` gerada.
2. `found` com token inválido → rejeitado.
3. `found` duas vezes → idempotente, retorna mesmos dados.
4. `isFounded` antes de fundar → false; após → true.
5. `generateOnboardingUrl` usa fragment (`#multiaddr=...`), não query.
6. Serviço exposto em `POST /genesis/found` com validação de token.
7. Flags `PROVISIONAL_SYSTEM_PEER` presente após gênese.
