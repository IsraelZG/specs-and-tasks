## 1. Objetivo
Implementar componente React `ThemeScope` no `packages/design-system`: wrapper que aplica atributo `data-ds-<scope>` no elemento DOM, permitindo que overrides de tema escopados funcionem via cascata CSS
*(extraído de T-016 §1; `docs/rfcs/design-system-proposal.md` D1)*.

### Contratos exatos
```tsx
// --- packages/design-system/src/lib/ThemeScope.tsx ---
import { ReactNode } from 'react';

export type ThemeScopeType = 'module' | 'page' | 'component';

export interface ThemeScopeProps {
  scope: ThemeScopeType;
  scopeId: string;
  children: ReactNode;
}

/** Wrapper que aplica data-ds-<scope>="<scopeId>" no elemento DOM. */
export function ThemeScope({ scope, scopeId, children }: ThemeScopeProps): JSX.Element;
```
Exemplo:
```tsx
<ThemeScope scope="module" scopeId="chat">
  <ThemeScope scope="page" scopeId="settings">
    <Button>CTA</Button>
  </ThemeScope>
</ThemeScope>
```
Renderiza:
```html
<div data-ds-module="chat">
  <div data-ds-page="settings">
    <button data-ds-component="cta">CTA</button>
  </div>
</div>
```
