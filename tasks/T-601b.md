# T-601b · Fork/Merge: resolveFork + createMergeNode

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (Node puro)

## 1. Objetivo
Implementar resolução determinística de fork e criação de nó de merge no pacote `@plataforma/core`
*(extraído de T-601 §1; depende de T-601a para detectar forks, T-103 para HLC)*.

### Contratos exatos
```ts
// --- packages/core/src/dag/merge.ts ---
import { SignedNode } from '../signature.js'; // T-107
import { StoragePort } from '@plataforma/protocol'; // T-004
import { ULID } from '../ulid.js'; // T-102

/** Resolve fork: nó com maior HLC vence; empate → maior pubkey (lexicográfica). */
export function resolveFork(forks: SignedNode[]): SignedNode;

/** Cria nó de merge apontando MUTATES para ambas as pontas do fork. */
export function createMergeNode(winner: SignedNode, loser: SignedNode, authorKey: Ed25519PrivateKey): Promise<SignedNode>;

/** Aplica merge: insere nó-merge no grafo. */
export function applyMerge(storage: StoragePort, forkNodes: SignedNode[]): Promise<ULID>;
```
