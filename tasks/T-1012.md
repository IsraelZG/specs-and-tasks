# T-1012 · Isolamento branch-por-task (git worktree)

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Test Runner:** `vitest`
- **Capacidade-alvo:** haiku *(escopo fechado; assinaturas e comandos git explícitos abaixo)*

## 1. Objetivo
Ao iniciar uma task (`start`), provisionar um `git worktree` na branch `task/<ID>` e
**persistir** `branch`/`worktreePath` no frontmatter da task, para isolamento e
paralelismo. Best-effort: se o git faltar ou falhar, NÃO quebra o `start`.

- **[CREATE]** `apps/nexus-backend/src/services/worktree.service.ts`
  ```ts
  export interface WorktreeInfo { branch: string; path: string }
  export interface WorktreeOptions { rootDir?: string; baseDir?: string }
  export async function ensureTaskWorktree(id: string, opts?: WorktreeOptions): Promise<WorktreeInfo>
  export async function removeTaskWorktree(id: string, opts?: WorktreeOptions): Promise<void>
  ```
  - `rootDir` default = `NEXUS_ROOT_DIR` ou `process.cwd()`. `baseDir` (onde criar os
    worktrees) default = `NEXUS_WORKTREE_DIR` ou `path.resolve(rootDir, '../.nexus-worktrees')`.
  - Usar SEMPRE `execFile('git', [...], { cwd: rootDir })` (NUNCA `exec` com string).
  - `ensureTaskWorktree`: branch = `task/${id}`, path = `path.join(baseDir, id)`.
    1. Se `path` já existe E aparece em `git worktree list --porcelain` → retorna `{branch,path}` (idempotente).
    2. Branch existe? `git rev-parse --verify --quiet refs/heads/task/${id}` (exit 0 = existe).
    3. Criar: se a branch existe → `git worktree add <path> task/${id}`; senão → `git worktree add <path> -b task/${id}`.
- **[UPDATE]** `task.service.ts` no `transition` quando `action==='start'`: APÓS gravar
  status/log, se `process.env.NEXUS_WORKTREE_ENABLED === '1'`, chamar
  `ensureTaskWorktree(id, { rootDir: this.rootDir })` dentro de try/catch (best-effort:
  em erro, só `console.warn`, não relança) e, em sucesso, `this.setMeta(id, info)`.
  (Gate por env para manter os testes da T-1010 inalterados por padrão.)
