- **Package Manager:** `pnpm`
- **Monorepo:** Turborepo

## 1. Objetivo
Implementar `validateEpochAlignment` no pacote `@plataforma/protocol`: ao receber `identity_epoch_index` divergente durante handshake Noise_XX (T-202), manter conexão e desviar para pipeline de catch-up (stub, ligado à Onda 0 no M3). Conexão NÃO é descartada
*(derivado de `docs/plano-de-implementacao.md` M2 — T-203; `docs/conceitos/noise-xx.md` — divergência de época → catch-up)*.

### Contratos exatos
```ts
// --- packages/protocol/src/epoch.ts ---

export interface EpochValidation {
  aligned: boolean;
  localIndex: number;
  remoteIndex: number;
  requiresCatchUp: boolean;
}

/** Compara identity_epoch_index local vs remoto. */
export function validateEpochAlignment(localIndex: number, remoteIndex: number): EpochValidation;

/** Sinaliza necessidade de catch-up (stub — Onda 0 no M3). */
export function signalEpochCatchUp(peerId: PeerId): void;
```
