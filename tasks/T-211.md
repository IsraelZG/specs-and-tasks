## 1. Objetivo
Implementar aba "Rede" no app Bancada (T-008): exibe `SwarmRegistry.connected()` com estado, latência e tier de cada peer, com botão simulado de desconexão. Como parte da Fase B (Bancada Visual), pode opcionalmente atuar como cliente da API `/control` do Hub (T-009).
*(derivado de T-008 §1 — "Bancada: cockpit de diagnóstico com abas"; T-205 §1 — SwarmRegistry em RAM)*.

### Contratos
```tsx
// --- apps/bancada/src/tabs/RedeTab.tsx 
---
import { SwarmRegistry } from '@plataforma/transport'; // T-205

export interface RedeTabProps {
  swarm: SwarmRegistry;
}

export function RedeTab({ swarm }: RedeTabProps): JSX.Element;
```
- Renderiza tabela/listagem de `swarm.all()` (todos os peers com state, latência, tier; connected primeiro), com filtro visual por estado de conexão.
- Cada linha: PeerId (truncado), ConnectionState (ícone colorido), latência (ms), tier, personas count.
- Botão "Derrubar" por peer → `swarm.remove(peerId)`.
- Atualização: re-renderiza a cada 5s ou via evento do swarm (subscrevendo com `swarm.onEvent()`).
