# T-1016 · Outlet de compressão + PromptBuilder religado + desacoplar legados

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm`
- **Test Runner:** `vitest` (backend) + `pnpm --filter nexus-frontend build` (frontend)
- **Capacidade-alvo:** sonnet *(mecânica; contratos já fixados por T-1018/T-1015)*

## 1. Objetivo
Conserta o travamento do PromptBuilder e entrega o "outlet" de compressão: um
endpoint/tool que comprime texto colado (para encurtar descrições de skills/agents,
ou qualquer contexto) via a interface `Compressor` (T-1018). Remove SmolLM/TurboVec/
EpochDB do **caminho crítico** (arquivos ficam no git, reintroduzíveis pelos gatilhos).

## 3. Escopo de Arquivos (Inputs e Outputs) — contratos FIXOS
- **[UPDATE]** `apps/nexus-backend/src/index.ts`:
  - **Remover** imports/uso de `./services/router.js`, `./services/turbovec.client.js`,
    `./services/headroom.client.js` e a rota `/api/build-prompt`.
  - **Adicionar** `POST /api/compress`, body `{ text: string }` → 200 com
    `CompressionResult` (`{ compressed, stats }`) usando `getCompressor()`; `text`
    ausente/{não-string} → 400.
- **[UPDATE]** `apps/nexus-backend/src/mcp/server.ts`:
  - **Remover** a tool `nexus_read_context` e os imports de `routeIntent`,
    `TurboVecClient`, `HeadroomClient`, `EpochDBClient`.
  - **Adicionar** a tool `nexus_compress_text` (input `{ text: string }`) → delega a
    `getCompressor().compress(text)`; seguir o padrão de delegação/literal já usado.
    (Manter `nexus_run_safe_script`, as task tools e `nexus_build_export` da T-1015.)
- **[UPDATE]** `apps/nexus-frontend/src/api.ts` — adicionar
  `compressText(text: string): Promise<{ compressed: string; stats: { originalChars: number; compressedChars: number; ratio: number; engine: string } }>`
  (POST `/api/compress`).
- **[UPDATE]** `apps/nexus-frontend/src/pages/PromptBuilder.tsx` — reescrever como
  outlet: `<textarea>` "cole o texto" → botão chama `api.compressText` → mostra o
  texto comprimido, `stats` (original→comprimido, %), e botão "Copiar". Usar
  `response.ok` (já no client), estados de loading/erro, e `<label htmlFor>` no campo.
- **[CREATE]** `apps/nexus-backend/src/tests/compress.api.test.ts` (rota `/api/compress`).
