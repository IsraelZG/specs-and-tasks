## 1. Objetivo
Implementar canal do Private Swarm no pacote `@plataforma/transport`: `Device_Sync_Key = HKDF(master, "device-sync-v1")`; `RendezvousId = blake2s256(Device_Sync_Key)`; documento casca dedicado; E2E com Device_Sync_Key; fora do RBSR principal; prioridade entre background e aba ativa
*(derivado de `docs/plano-de-implementacao.md` §10 T-702)*.

### Contratos exatos
```ts
// --- packages/transport/src/privateSwarm/privateSwarm.ts 
---
import { PeerId } from '@plataforma/protocol';

export interface PrivateSwarmConfig {
  masterKey: Uint8Array;          // chave mestra do dispositivo
  maxPeers: number;               // default 10 (dispositivos do mesmo usuário)
  priorityMode: 'background' | 'active';
}

export interface PrivateSwarmChannel {
  channelId: string;              // RendezvousId
  syncKey: Uint8Array;            // Device_Sync_Key (32 bytes)
  connectedDevices: PeerId[];
  createdAt: number;
}

export interface PrivateSwarmMessage {
  type: 'rascunho' | 'preferencia' | 'prefetch_cache';
  payload: Uint8Array;            // cifrado com Device_Sync_Key
  senderDeviceId: PeerId;
  timestamp: number;
  hlc: number;
}

export class PrivateSwarm {
  constructor(config: PrivateSwarmConfig);

  /** Inicializa canal com Device_Sync_Key. */
  initChannel(): Promise<PrivateSwarmChannel>;

  /** Conecta novo dispositivo ao private swarm. */
  connectDevice(devicePeerId: PeerId): Promise<void>;

  /** Envia mensagem cifrada E2E. */
  sendMessage(msg: PrivateSwarmMessage): Promise<void>;

  /** Recebe mensagens do private swarm. Retorna função de unsubscribe. */
  onMessage(handler: (msg: PrivateSwarmMessage) => void): () => void;

  /** Define prioridade (background vs aba ativa). */
  setPriority(mode: 'background' | 'active'): void;

  /** Verifica que RendezvousId não permite derivar a chave mestra. */
  static verifyDerivationUnidirectionality(masterKey: Uint8Array): boolean;
}
```
- `Device_Sync_Key = HKDF(master, "device-sync-v1")` — derivação unidirecional.
- `RendezvousId = blake2s256(Device_Sync_Key)` — observação do RendezvousId não permite derivar a mestra.
- Canal fora do RBSR principal: usa documento casca Automerge dedicado (T-403).
- E2E: todas as mensagens cifradas com `Device_Sync_Key`.
- Tráfego ilegível para o relay (asserção).
- Prioridade: background (baixa frequência) vs aba ativa (tempo real).
