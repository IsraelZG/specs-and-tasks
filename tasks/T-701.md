## 1. Objetivo
Implementar `LinkCipher` no pacote `@plataforma/transport`: cifra/decifra frames com AES-256-GCM usando chaves de sessão derivadas do handshake Noise_XX
*(derivado de T-202 — Noise_XX handshake gera chaves de sessão; T-101 — AES-256-GCM wrappers)*.

### Contratos exatos
```ts
// --- packages/transport/src/crypto/LinkCipher.ts ---
import { AesKey } from '@plataforma/crypto'; // T-101

/** Cifra frame para transmissão no link. */
export function encryptFrame(sessionKey: AesKey, plaintext: Uint8Array): Promise<Uint8Array>;

/** Decifra frame recebido. */
export function decryptFrame(sessionKey: AesKey, ciphertext: Uint8Array): Promise<Uint8Array>;

/** Deriva chave de sessão AES a partir das chaves pós-Noise. */
export function deriveSessionKey(handshakeKey: Uint8Array, nonce: Uint8Array): Promise<AesKey>;
```
- `encryptFrame`: AES-256-GCM, gera nonce interno (12 bytes), retorna `nonce || ciphertext || authTag`.
- `decryptFrame`: extrai nonce, decifra, valida authTag.
- Cifra apenas cabeçalhos e metadados de transporte; payload do grafo já é assinado (Layer 1).
