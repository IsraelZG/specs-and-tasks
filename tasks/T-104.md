## 1. Objetivo
Implementar geração de mnemônico BIP39 (12/24 palavras) + derivação determinística da chave mestra Ed25519 + derivação de chave de dispositivo via PBKDF2 da senha, no pacote `@plataforma/crypto`
*(derivado de `docs/plano-de-implementacao.md` §4 T-104; `docs/visao-arquitetural.md` §7 — "A chave mestra nunca sai do dispositivo; dispositivos falam por personas via delegação")*.

### Contratos exatos (assinaturas TS fixadas)

```ts
// --- packages/crypto/src/bip39.ts ---
import { Ed25519PrivateKey, Ed25519PublicKey } from './wrappers.js'; // T-101

/** Mnemônico BIP39 como string de palavras separadas por espaço. */
export type Mnemonic = string;

/** Entropia bruta (16 bytes = 128 bits para 12 palavras; 32 bytes = 256 bits para 24 palavras). */
export type Entropy = Uint8Array;

/** Gera mnemônico BIP39 de 12 palavras (128 bits de entropia). */
export function generateMnemonic(): Mnemonic;

/** Gera mnemônico de 24 palavras. */
export function generateMnemonic24(): Mnemonic;

/** Valida checksum BIP39. */
export function validateMnemonic(mnemonic: Mnemonic): boolean;

/** Deriva seed (64 bytes) do mnemônico + passphrase opcional via PBKDF2 (2048 iterações, HMAC-SHA512).
 *  BIP39 §"From Mnemonic to Seed". */
export function mnemonicToSeed(mnemonic: Mnemonic, passphrase?: string): Promise<Uint8Array>;

/** Deriva chave mestra Ed25519 a partir do seed (usa primeiros 32 bytes como private key). */
export function seedToMasterKey(seed: Uint8Array): Promise<{ publicKey: Ed25519PublicKey; privateKey: Ed25519PrivateKey }>;

/** Deriva chave do dispositivo via PBKDF2(senha, salt, 100_000 iterações, 32 bytes).
 *  Esta chave cifra localmente a master key; NÃO é a chave de identidade do grafo. */
export function deriveDeviceKey(password: string, salt: Uint8Array): Promise<Uint8Array>;
```

### Decisão de escopo (resolvida)
T-101 (Wrappers cripto) deixou BIP39/PBKDF2 como ABERTO. **Esta task (T-104) encapsula toda a lógica BIP39 + derivação.** T-101 provê apenas os wrappers Ed25519 e hashes.

Casos de teste:
1. `generateMnemonic()` retorna 12 palavras válidas; `validateMnemonic` → true.
2. `generateMnemonic24()` retorna 24 palavras válidas.
3. `validateMnemonic` com palavra trocada → false.

8. `deriveDeviceKey` com mesma senha+salt → mesma chave; senhas diferentes → chaves diferentes.
