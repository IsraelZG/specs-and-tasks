## 1. Objetivo
Implementar protocolo de troca RBSR no pacote `@plataforma/protocol`: máquina de estados do reconciler (Root → Sub-ranges → Request → Apply), usando B-Tree de fingerprints (T-301) e NetworkAdapterPort (T-204) para reconciliar grafos entre dois peers
*(derivado de `docs/visao-arquitetural.md` §8 — "três ondas: anti-entropy → tela ativa → histórico em background"; T-301 — RBSRBTree; T-204 — WebSocket adapter)*.

### Contratos exatos
```ts
// --- packages/protocol/src/rbsr/exchange.ts ---
import { RBSRBTree, Fingerprint, RangeDiff } from './BTree.js'; // T-301
import { NetworkAdapterPort, PeerId, WireData } from '../ports.js'; // T-004

export enum RBSRState {
  IDLE          = 'idle',
  ROOT_EXCHANGE = 'root_exchange',
  RANGE_COMPARE = 'range_compare',
  REQUEST_NODES = 'request_nodes',
  APPLY         = 'apply',
  DONE          = 'done',
}

export interface RBSRSession {
  localTree: RBSRBTree;
  remotePeerId: PeerId;
  transport: NetworkAdapterPort;
  state: RBSRState;
}

/** Inicia reconciliação: troca root fingerprints, compara ranges, baixa nós divergentes. */
export async function reconcile(session: RBSRSession): Promise<number>; // retorna qtd de nós importados

/** Envia root fingerprint para peer remoto. Frame RBSR_ROOT. */
export function sendRootFingerprint(transport: NetworkAdapterPort, peerId: PeerId, rootXor: Fingerprint): Promise<void>;

/** Compara ranges e envia pedido de nós ausentes. */
export function requestNodes(transport: NetworkAdapterPort, peerId: PeerId, diff: RangeDiff): Promise<void>;

/** Aplica nós recebidos no storage local, validando Layer 1 (T-107). */
export function applyNodes(nodes: Uint8Array[], storage: StoragePort): Promise<number>;
```
- Máquina de estados: `IDLE` → `ROOT_EXCHANGE` (compara rootXor) → se igual → `DONE`; se diferente → `RANGE_COMPARE` (split recursivo) → `REQUEST_NODES` (baixa nós divergentes) → `APPLY` (valida + insere).
- Três ondas: anti-entropy O(1) (root), tela ativa (ranges divergentes), histórico em background (ranges restantes).

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro).
- [x] **Ambiente:** `pnpm --filter @plataforma/protocol test`. Usar SimNetwork (T-005) como transporte.

Casos de teste:
1. 2 peers com árvores idênticas → `reconcile` retorna 0 nós importados (DONE imediato).
2. Peer A tem 3 nós extras → `reconcile` importa 3 nós em B.
3. Peer A e B têm 1 nó com mesmo id mas fingerprint diferente (conflito) → detectado, não sobrescreve cegamente.
4. `reconcile` sob partição de rede → timeout, retorna partial.
5. `applyNodes` rejeita nós com assinatura Layer 1 inválida.
6. 50 nós divergentes → reconciliados em < 5 rounds de split.
