# T-201b · Wire format: decodeFrame + quarentena

## 0. Ambiente de Execução Obrigatório

- **Package Manager:** `pnpm`

## 1. Objetivo
Implementar `decodeFrame` e `tryReadFrame` no pacote `@plataforma/protocol`: decodificação de frame binário com regras de quarentena para versão futura, LENGTH inválido, e payload malformado. Nunca lança exceção
*(extraído de T-201 §1; derivado de `docs/caderno-2-protocol/05-wire-protocol.md` §3)*.

### Contratos exatos
```ts
// --- packages/protocol/src/wireFormat.ts (extensão) ---
import { FrameType, WireFrame } from './wireFormat.js'; // T-201a

export type DecodeResult =
  | { ok: true; frame: WireFrame }
  | { ok: false; reason: 'too_large' | 'bad_length' | 'unknown_version' | 'parse_error'; retentionState?: number; raw?: Uint8Array };

/** Decodifica frame binário. Nunca lança. */
export function decodeFrame(buffer: Uint8Array): DecodeResult;

/** Lê frame de stream incremental. Retorna {frame, consumed} ou null se incompleto. */
export function tryReadFrame(buffer: Uint8Array): { frame: DecodeResult; consumed: number } | null;
```

Regras de quarentena (`caderno-2-protocol/05-wire-protocol.md` §3):
- Versão > WIRE_VERSION + tipo volátil (EPHEMERAL, HEARTBEAT_*) → descartar.
- Versão > WIRE_VERSION + tipo auditável → `{ok:false, retentionState:3, raw}`.
- LENGTH > MAX_FRAME_LENGTH → `{ok:false, reason:'too_large'}`.

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro).

Casos de teste:
1. `encodeFrame` + `decodeFrame` roundtrip → `{ok:true}`.
2. Frame vazio (0 bytes) → `{ok:false, reason:'bad_length'}`.
3. LENGTH > 1 MiB → `{ok:false, reason:'too_large'}`.
4. Versão `0x00` → `{ok:false, reason:'unknown_version'}`.
5. Versão `0xFF` + tipo RBSR_NODES (auditável) → `{ok:false, retentionState:3}`.
6. Versão `0xFF` + tipo HEARTBEAT_PING (volátil) → `{ok:false}`, sem retentionState.
7. Payload MessagePack inválido → `{ok:false, reason:'parse_error'}`.
8. Fuzz: 1000 frames aleatórios → `decodeFrame` nunca lança.
9. `tryReadFrame` com buffer parcial (3 bytes) → `null`.
