# T-501 · Motor de UCAN Core

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v20+
- **Package Manager:** `pnpm` (NÃO USE npm ou yarn)
- **Monorepo:** Turborepo
- **Test Runner:** `vitest` (Node puro)
- **Capacidade-alvo:** sonnet

## 1. Objetivo
Implementar motor de UCAN (emissão, validação, delegação) no pacote `@plataforma/core`, baseado em assinaturas Ed25519
*(derivado de `docs/visao-arquitetural.md` §7 — "UCAN é permissão de requisição"; T-101 — Ed25519; T-107 — assinatura canônica)*.

### Contratos exatos
```ts
// --- packages/core/src/auth/ucan.ts ---
import { Ed25519PublicKey, Ed25519Signature } from '@plataforma/crypto'; // T-101
import { ULID } from '../ulid.js'; // T-102

export interface UcanPayload {
  iss: ULID;            // issuer (quem emite)
  aud: ULID;            // audience (quem recebe a delegação)
  att: UcanCapability[];// attenuation (capacidades delegadas)
  exp: number;          // expiration (timestamp ms)
  nbf?: number;         // not before
  nnc?: string;         // nonce
  fct?: unknown[];      // facts
}

export interface UcanCapability {
  with: string;         // recurso (ex: "entity:<entityId>")
  can: string;          // ação (ex: "READ", "WRITE", "DELEGATE")
  nb?: Record<string, unknown>; // constraints
}

export interface SignedUcan {
  header: { alg: 'EdDSA'; typ: 'JWT' };
  payload: UcanPayload;
  signature: Ed25519Signature;
}

/** Emite UCAN assinado com chave privada do issuer. */
export function issueUcan(
  issuerKey: Ed25519PrivateKey,
  audiencePubKey: Ed25519PublicKey,
  capabilities: UcanCapability[],
  ttlMs?: number, // default 4h
): Promise<SignedUcan>;

/** Valida UCAN: assinatura, expiração, atenuação (não pode ampliar escopo). */
export function verifyUcan(ucan: SignedUcan, rootIssuerPubKey: Ed25519PublicKey): Promise<boolean>;

/** Verifica se UCAN filho é atenuação válida do pai (subset de capacidades). */
export function isAttenuation(child: UcanPayload, parent: UcanPayload): boolean;

/** Serializa UCAN para transmissão (JWT compact). */
export function encodeUcan(ucan: SignedUcan): string;

/** Desserializa UCAN. */
export function decodeUcan(token: string): SignedUcan;
```

## 4. Estratégia de Testes
- [x] **Framework:** Vitest (Node puro).
- [x] **Ambiente:** `pnpm --filter @plataforma/core test`.

Casos de teste:
1. `issueUcan` + `verifyUcan` roundtrip.
2. `verifyUcan` com assinatura adulterada → false.
3. `verifyUcan` com UCAN expirado → false.
4. `isAttenuation`: filho com subset de `can` → true.
5. `isAttenuation`: filho com `can` não presente no pai → false (não pode ampliar escopo).
6. `encodeUcan` + `decodeUcan` roundtrip.
7. Delegação em cadeia: A→B→C, C usa UCAN de B que foi delegado por A.

## 5. Instruções de Execução

3. Implemente `verifyUcan` (signature + expiration + attenuation).
4. Implemente `isAttenuation` (subset check).
5. Re-exporte. Rode build + test.
