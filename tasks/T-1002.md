# T-1002 · Integração do Indexador Semântico (TurboVec)

## 0. Ambiente de Execução Obrigatório
- **Runtime:** Node.js v22+
- **Package Manager:** `pnpm`
- **Test Runner:** `vitest`

## 1. Objetivo
Implementar um serviço no backend Node (`apps/nexus-backend`) que faz o parse dos arquivos Markdown da pasta `docs/` e `tasks/`, extrai o *frontmatter* YAML e indexa os conteúdos utilizando o TurboVec.

## 5. Instruções de Execução (Step-by-Step)

3. Escreva os pedaços (chunks) junto com seus metadados na interface do TurboVec.
