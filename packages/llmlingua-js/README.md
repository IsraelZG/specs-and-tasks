# LLMLingua-JS\n\nImplementação em TypeScript puro do algoritmo LLMLingua para compressão de prompts usando Transformers.js.\n\n## Instalação\n\n```bash\nnpm install\n```\n\n## Exemplo de Uso\n\n```typescript\nimport { LLMLingua } from './src/index';\nconst compressor = new LLMLingua();\nconst result = await compressor.compress('Seu prompt longo aqui...');\nconsole.log(result);\n```\n\n## Como funciona\nO LLMLingua utiliza a perplexidade de um modelo causal (SmolLM2) para identificar tokens de baixa informação e removê-los, mantendo a semântica do prompt original enquanto reduz drasticamente o tamanho do contexto.
