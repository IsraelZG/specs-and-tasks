# Utilitário de IA

## Definição

Todo modelo (LLM, embedding, transcrição, classificação, visão) é uma capacidade `compute` da RFC-010: contrato tipado, flag de determinismo (IA = não-determinística), classe de privacidade. IA não é privilégio especial no protocolo. O contrato completo está em `caderno-3-sdk/14-ia-rag-e-agentes.md §1`.
