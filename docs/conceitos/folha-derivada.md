# Folha Derivada

## Definição

A **folha derivada** é a apuração da folha de pagamento como lançamentos contábeis e tributários derivados dos eventos de trabalho dos colaboradores (como ponto, hora extra, admissão) pela SPEC jurisdicional na competência.

## Contrato

As especificações e tratamento de privacidade de dados de RH estão documentadas em [[caderno-3-sdk/17-contabil-fiscal-rh-reference-spec#5-rh-e-folha]].
