# Classe de Liquidação

## Definição

A **classe de liquidação** declarada na SPEC de um item determina o que a operação de compra (`CREDITS`) materializa para o comprador (por exemplo: transferência de inventário escasso, concessão de permissão, ativação de assinatura, execução de serviço).

## Contrato

As classes de liquidação canônicas suportadas estão detalhadas em [[caderno-3-sdk/15-marketplace-reference-spec#2-classes-de-liquidação]].
