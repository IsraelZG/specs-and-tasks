# Lançamento Derivado

## Definição

Um **lançamento derivado** é a classificação contábil (débito/crédito em contas) gerada por uma SPEC contábil a partir de fatos econômicos existentes no grafo (como vendas, baixas, pagamentos), sem necessidade de redigitação. O razão contábil é derivado e reconstrutivo da linhagem dos fatos econômicos.
