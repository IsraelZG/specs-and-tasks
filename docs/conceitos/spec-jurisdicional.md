# SPEC Jurisdicional

## Definição

A SPEC base declara os invariantes universais do domínio. Variantes jurisdicionais estendem a base via `EXTENDS`, carregando tabelas e procedimentos Zen daquela região. A variante só pode especializar, nunca contradizer os invariantes da base. O contrato completo está em `caderno-3-sdk/13-jurisdicao.md §2`.
