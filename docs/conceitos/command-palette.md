# Command Palette

## Definição

A command palette (atalho Cmd/Ctrl-K) é a superfície única de intenção em linguagem natural. Resolve para: busca (recuperação híbrida), ação (intent) ou geração (agente de IA produz SPEC:PAGE/SPEC:WORKFLOW). A superfície é da RFC-026 (Shell); o comportamento de IA é da RFC-011. O contrato completo está em `caderno-3-sdk/14-ia-rag-e-agentes.md §7`.
