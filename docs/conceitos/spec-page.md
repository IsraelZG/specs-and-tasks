# SPEC:PAGE (Página)

## Definição

Uma página é um nó **`SPECIFICATION` (kind: `PAGE`)** cujo payload é um documento do dialeto da linguagem de páginas (RFC-008). Herda linhagem, assinatura, governança e distribuição por replicação normal do grafo. O contrato completo está em `caderno-3-sdk/11-linguagem-de-paginas.md`.
