# Mockups & ferramentas de design

Protótipos HTML standalone (abrem direto no navegador; tokens de tema claro/escuro embutidos,
aproximando o design-system). São **referência de design** para anexar às tasks de UI — não código de produção.

| Arquivo | O quê | Task(s) de UI |
|---|---|---|
| [onboarding-boas-vindas.html](onboarding-boas-vindas.html) | Tela de boas-vindas / escolha de entrada (criar rede · convite · identidade) | T-512, T-208/209/210 |
| [gerador-nome-recursivo.html](gerador-nome-recursivo.html) | Ferramenta: gera siglas recursivas (1ª letra recursiva + 2 letras que expandem por domínio) para o nome da rede | — (decisão de marca) |

Convenções aplicadas (ver `../diretrizes-ux.md`): só tokens `--ds-*`, i18n visível, a11y por teclado,
sentence case, sem cor literal. Inventário completo de telas em `../inventario-de-telas.md`.
